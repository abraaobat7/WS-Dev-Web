Referência Visual de Propriedades CSS de Texto

Cada card mostra o efeito visual e a linha de código CSS correspondente.

1. Font Family (Família da Fonte)

Serif

Texto em fonte serifada.

.exemplo { font-family: Georgia, serif; }
Sans-serif

Texto em fonte sem serifa.

.exemplo { font-family: 'Inter', sans-serif; }
Monospace

const x = 10;

.exemplo { font-family: 'JetBrains Mono', monospace; }

9. Truncation (Truncamento)

1 Linha com Reticências

Este é um texto muito longo que será truncado e terminará com reticências.

.exemplo { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }
Múltiplas Linhas (Clamp)

11. Pseudo-elementos

Primeira Linha

A primeira linha deste texto fica em negrito e maiúscula. As demais linhas continuam normais.

.exemplo::first-line { font-weight: bold; text-transform: uppercase; }
Primeira Letra (Drop Cap)

A primeira letra fica grande e estilizada, como em livros e revistas.

.exemplo::first-letter { font-size: 3em; float: left; }
